Description

This project relates to information security in that when you browse the internet, cookies store your personal information and where you visited. The goal of this project was to give the user of the extension a little more control over who has access to that information. We set out 2 main functional objectives:

  1. Popup: Have a popup to let you block and delete cookies most recently added
  2. Extension's Page: Allow blocking, unblocking, deleting, and spoofing of any cookie stored in the browser

The popup is to allow the user to on the fly delete or block unwanted cookies recently stored on the browser. This allows for a quicker way to navigate the internet, rather than going to the extension page each time. The extension page itself though has the main functionality control for the extension. Here you can view a list of blocked and unblocked domains, as well as delete and modify currently stored cookie data. You can finally view Cookie Analytics. This tab allows you to view which domains store 3rd party cookies on your browser. Enjoy the extension and safe browsing!
